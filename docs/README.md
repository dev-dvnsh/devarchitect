# Devarchitect

A CLI-based, AI-assisted project architecture and planning tool for solo
developers and small teams.

## The Problem

Developers fail projects not because they lack skills, but because the
roadmap is foggy. Tools like Git track _what_ changed in code — but nobody
tracks _why_ decisions were made, what the vision is, or how the project
is supposed to evolve.

## What Devarchitect Does

Devarchitect fills that gap. It stores your project's engineering memory —
vision, decisions, architecture, roadmap, and progress — as versioned JSON
files inside a `.devarchitect/` folder that lives alongside your code.

Any new developer joining the project can run one command and immediately
understand:

- What the project is and why it exists
- What decisions were made and why
- Where the project currently stands
- What the next steps are

## Who It's For

Solo developers and small teams who want structured, AI-guided planning
without the overhead of heavy project management tools.
